# Claude Code Configuration

This project is fully configured for Claude Code CLI with instructions, slash commands, and checklists.

## Quick Start

Claude Code automatically reads `CLAUDE.md` for project context. Just run:

```bash
claude
```

## Project Instructions

**Main file:** `CLAUDE.md` - Project overview, architecture, and coding standards

**Detailed instructions:** `.github/instructions/` - 29 comprehensive instruction files covering:
- Architecture, Frontend, Backend, API patterns
- Testing, Security, Performance, Docker
- Database, Git workflow, CI/CD

## Slash Commands (Specialized Agents)

Use these commands to invoke specialized expertise:

| Command | Description |
|---------|-------------|
| `/react` | React 19 frontend development |
| `/api` | API architecture and design |
| `/debug` | Systematic debugging process |
| `/db` | PostgreSQL & Prisma database operations |
| `/devops` | Docker and infrastructure |
| `/test` | Playwright E2E testing |
| `/a11y` | Accessibility (WCAG) compliance |
| `/cleanup` | Code cleanup and refactoring |
| `/actions` | GitHub Actions workflows |
| `/typescript` | Advanced TypeScript patterns |
| `/ux` | UX/UI design patterns |

### Usage Examples

```bash
# Get React help
claude "/react help me create a media player component"

# Debug an issue
claude "/debug the player stops after each song"

# Design an API
claude "/api design the playlist endpoints"

# Database help
claude "/db optimize the media queries"
```

## Checklists

Before completing tasks, reference the appropriate checklist in `.ai/checklists/`:

- `api-endpoint.md` - Creating/updating API endpoints
- `react-component.md` - Creating/updating React components
- `database-change.md` - Database schema changes
- `feature-complete.md` - Before merging features

## Copilot Compatibility

This project works with both:
- **Claude Code CLI:** Uses `CLAUDE.md` + `.claude/commands/`
- **GitHub Copilot CLI:** Uses `.github/instructions/` + `.github/copilot/agents/`

Both tools follow the same patterns and standards!
